3. SPRINT PLAN
----------------------------------------------------------------

SPRINT 1 (Weeks 1-2): Data Collection & Preprocessing
------------------------------------------------------
Goal     : Build a clean, structured dataset ready for modeling
Stories  : US-01, US-02

Tasks:
  - Download H1B visa dataset from Kaggle (857,216 records)
  - Identify and drop irrelevant columns
  - Handle missing values (median for PREVAILING_WAGE, mode for YEAR)
  - Fill categorical nulls with 'UNKNOWN'
  - Generate synthetic target variable: processing_time_days (range: 15-120)
  - Save cleaned dataset to CSV

Definition of Done:
  - preprocess_visa.py runs without errors
  - Cleaned dataset has no null values
  - Target variable generated and validated

Sprint 1 Status: COMPLETED ✅

---

SPRINT 2 (Weeks 3-4): Exploratory Data Analysis & Feature Engineering
----------------------------------------------------------------------
Goal     : Derive insights and prepare features for modeling
Stories  : US-03, US-04

Tasks:
  - Visualize processing time distribution (histogram)
  - Analyze wage vs processing time (scatter plot)
  - Boxplot of processing time by year
  - Extract STATE from WORKSITE column (dimensionality reduction)
  - Create WAGE_CATEGORY feature (binned: Very Low to Very High)
  - Create FULL_TIME_BINARY feature
  - Apply frequency encoding for EMPLOYER_NAME and JOB_TITLE
  - Apply LabelEncoder for categorical columns

Definition of Done:
  - milestone2.py runs without errors
  - All visualizations saved as screenshots
  - Final feature set ready for model training

Sprint 2 Status: COMPLETED ✅

---

SPRINT 3 (Weeks 5-6): Predictive Modeling
------------------------------------------
Goal     : Train and evaluate XGBoost regression model
Stories  : US-05, US-06

Tasks:
  - Set up train/test split (80/20)
  - Train XGBoost Regressor (300 estimators, max_depth=6, lr=0.05)
  - Evaluate using MAE, RMSE, R² metrics
  - Save model and encoders using joblib
  - Store feature importances alongside model

Results:
  - MAE  : 5.00 days
  - RMSE : 5.77 days
  - R²   : 0.6267

Definition of Done:
  - Model achieves MAE < 10 days
  - visa_model.pkl saved successfully
  - Metrics printed and recorded

Sprint 3 Status: COMPLETED ✅

---

SPRINT 4 (Weeks 7-8): Web App Development & Deployment
-------------------------------------------------------
Goal     : Build and deploy a functional Streamlit estimator app
Stories  : US-07, US-08, US-09, US-10

Tasks:
  - Build Streamlit app with 3 tabs (Predict, Analytics, Model Info)
  - Create input form (case status, wage, year, state, job title etc.)
  - Integrate model for real-time prediction
  - Add gauge chart for visual output
  - Add confidence range (±15%)
  - Add trend charts using Plotly
  - Add feature importance bar chart
  - Deploy on Streamlit Cloud
  - Push all code to GitHub

Definition of Done:
  - App runs locally without errors
  - App deployed and publicly accessible via URL
  - GitHub repo updated with all milestone code

Sprint 4 Status: COMPLETED ✅
